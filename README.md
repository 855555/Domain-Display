# 域名展示系统

![](/2025_134943.webp)

![](/2025_13507.webp)

这是一个简单而强大的域名展示系统，用于管理和展示域名集合。系统提供了直观的用户界面！

## 功能特点

- **域名展示**：以卡片形式展示域名信息，包括域名名称、TLD、提供商和到期日期
- **响应式设计**：适配各种屏幕尺寸，提供良好的移动端体验
- **备份功能**：直接备份，下次直接导入，无需再手动输入域名信息
- **购买功能**：提供域名购买入口

## 默认密码

```bash
admin123
```

## 如何安装

- vercel部署

- 构建命令 
```
bun run build
```

- 安装命令
```
bun install
```
